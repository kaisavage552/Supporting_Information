# Surfaces and Wulff Construction

Surface energy calculations for five Mg–Zn phases across multiple crystallographic orientations, and Wulff equilibrium crystal shape construction for each phase.

---

## Structure

```
Surfaces_and_Wulff/
├── mg_surface_energy/
├── mgzn2_surface_energy/
├── mg2zn11_surface_energy/
├── mg4zn7_surface_energy/
├── mg21zn25_surface_energy/
│
│   Each phase directory contains:
│   ├── bulk_structure/          Relaxed CIF and LAMMPS data file (input to slab generation)
│   ├── slabs/                   ASE-format slab structures for each Miller index
│   ├── lammps_calculations/     LAMMPS input and output files for each surface calculation
│   ├── Notebooks/               Jupyter notebooks for slab generation and surface energy extraction
│   └── Wulff plot/              Notebook generating the per-phase Wulff shape
│
└── Wulff Plot report figure/
    ├── wulff_all_phases.png     Combined Wulff shapes used in the thesis
    └── (combined notebook)
```

---

## Workflow

### 1. Slab generation
Run the notebooks in `Notebooks/` for each phase. They use ASE to:
- Read the relaxed bulk structure from `bulk_structure/`
- Build symmetric slabs for each surface orientation (various Miller indices)
- Write slab geometries to `slabs/` and generate LAMMPS input files in `lammps_calculations/`

### 2. Surface energy calculation
```bash
lmp -in lammps_calculations/<surface>.data
```
Each calculation relaxes the slab and records the total energy. Surface energy is computed as:

> γ = (E_slab − N · E_bulk/atom) / (2 · A)

where A is the surface area and the factor of 2 accounts for both surfaces of the symmetric slab.

### 3. Wulff construction
The `Wulff plot/` notebook reads the computed surface energies and uses the Wulff construction algorithm (via `from pymatgen.analysis.wulff import WulffShape`) to generate the equilibrium crystal shape.

## Phases and Surfaces Studied

| Phase | Crystal system | Miller indices included |
|-------|---------------|------------------------|
| Mg (hcp) | Hexagonal | (0001), (10-10), (10-11), (10-12), (11-20) |
| MgZn₂ (C14) | Hexagonal | (0001), (10-10), (10-11), (11-20) |
| Mg₂Zn₁₁ | Cubic | (100), (110), (111), (210) |
| Mg₄Zn₇ | Monoclinic | (001), (010), (100), (101), (110) |
| Mg₂₁Zn₂₅ | Trigonal | (001), (100), (101), (110), (201) |
